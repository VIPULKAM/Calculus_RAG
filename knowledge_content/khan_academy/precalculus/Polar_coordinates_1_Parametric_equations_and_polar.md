# Polar coordinates 1 | Parametric equations and polar coordinates | Precalculus | Khan Academy

*Source: [Khan Academy Video](https://www.youtube.com/watch?v=jexMSlSDubM)*

## Key Concepts
- Introduction to polar coordinates as an alternative to Cartesian coordinates
- Converting between Cartesian coordinates (x, y) and polar coordinates (r, θ)
- Using trigonometry (SOHCAHTOA) and Pythagorean theorem for coordinate conversion

## Definitions
- **Cartesian coordinates**: A system that specifies points using horizontal (x) and vertical (y) distances from the origin. Notation: $(x, y)$
- **Polar coordinates**: A system that specifies points using distance from the origin (r) and angle from the positive x-axis (θ). Notation: $(r, θ)$
- **Conversion formulas**:
  - $r^2 = x^2 + y^2$ (Pythagorean theorem)
  - $\tan θ = \frac{y}{x}$ (Tangent ratio)

## Examples & Steps

**Example 1: Convert Cartesian coordinates (3, 4) to polar coordinates**

**Step 1: Find r (distance from origin)**
- Use Pythagorean theorem: $r^2 = 3^2 + 4^2$
- $r^2 = 9 + 16 = 25$
- $r = \sqrt{25} = 5$

**Step 2: Find θ (angle from x-axis)**
- Use tangent ratio: $\tan θ = \frac{y}{x} = \frac{4}{3}$
- $θ = \arctan\left(\frac{4}{3}\right)$
- Using calculator: $θ ≈ 53.13^\circ$

**Result**: The polar coordinates are $(5, 53.13^\circ)$

**General Conversion Formulas**
- **Cartesian to polar**: 
  - $r = \sqrt{x^2 + y^2}$
  - $θ = \arctan\left(\frac{y}{x}\right)$
- **Polar to Cartesian**:
  - $x = r \cos θ$
  - $y = r \sin θ$

## Summary
Polar coordinates provide an alternative way to describe points in a plane using distance and angle rather than horizontal and vertical positions. The conversion between Cartesian and polar coordinates relies on the Pythagorean theorem and trigonometric ratios. This system is particularly useful for describing circular and rotational patterns.
