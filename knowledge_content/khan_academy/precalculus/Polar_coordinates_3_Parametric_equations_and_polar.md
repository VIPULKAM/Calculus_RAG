# Polar coordinates 3 | Parametric equations and polar coordinates | Precalculus | Khan Academy

*Source: [Khan Academy Video](https://www.youtube.com/watch?v=9iqN12hCn10)*

## Key Concepts
- Converting equations between Cartesian coordinates ($x$ and $y$) and polar coordinates ($r$ and $\theta$)
- Using trigonometric relationships to substitute variables
- Simplifying equations through algebraic manipulation (factoring, dividing, squaring)

## Definitions
- **Cartesian coordinates**: Represent points using $(x, y)$
- **Polar coordinates**: Represent points using $(r, \theta)$, where:
  - $r$ = distance from origin
  - $\theta$ = angle from positive x-axis
- **Conversion formulas**:
  - $x = r\cos\theta$
  - $y = r\sin\theta$
  - $r^2 = x^2 + y^2$
  - $\sin\theta = \frac{y}{r}$
  - $\cos\theta = \frac{x}{r}$

## Examples & Steps

### Example 1: Cartesian to Polar (3y - 7x = 10)
1. Substitute polar equivalents:  
   $3(r\sin\theta) - 7(r\cos\theta) = 10$
2. Factor out $r$:  
   $r(3\sin\theta - 7\cos\theta) = 10$
3. Solve for $r$:  
   $r = \frac{10}{3\sin\theta - 7\cos\theta}$

### Example 2: Cartesian to Polar (y = 2x - 3)
1. Substitute polar equivalents:  
   $r\sin\theta = 2(r\cos\theta) - 3$
2. Rearrange terms:  
   $r\sin\theta - 2r\cos\theta = -3$
3. Factor out $r$:  
   $r(\sin\theta - 2\cos\theta) = -3$
4. Solve for $r$:  
   $r = \frac{-3}{\sin\theta - 2\cos\theta}$

### Example 3: Polar to Cartesian (r = 4sinθ)
1. Use $\sin\theta = \frac{y}{r}$:  
   $r = 4\left(\frac{y}{r}\right)$
2. Multiply both sides by $r$:  
   $r^2 = 4y$
3. Substitute $r^2 = x^2 + y^2$:  
   $x^2 + y^2 = 4y$

### Example 4: Polar to Cartesian (r = sinθ + cosθ)
1. Substitute trigonometric identities:  
   $r = \frac{y}{r} + \frac{x}{r}$
2. Multiply both sides by $r$:  
   $r^2 = y + x$
3. Substitute $r^2 = x^2 + y^2$:  
   $x^2 + y^2 = x + y$

### Example 5: Polar to Cartesian (r = a²)
1. Square both sides:  
   $r^2 = a^4$
2. Substitute $r^2 = x^2 + y^2$:  
   $x^2 + y^2 = a^4$

## Summary
This lesson demonstrated how to convert equations between Cartesian and polar coordinate systems using trigonometric substitutions and algebraic manipulation. The key steps involve replacing $x$ and $y$ with $r\cos\theta$ and $r\sin\theta$ when converting to polar form, and using identities like $r^2 = x^2 + y^2$ when converting to Cartesian form. These conversions allow the same relationships to be expressed in different coordinate systems.
